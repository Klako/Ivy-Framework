Official Auth0 authentication integration for the [Ivy Framework](https://ivy.app).

## Overview

`Ivy.Auth.Auth0` provides seamless Auth0 integration for user authentication and identity management within your Ivy applications.

Ivy is a modern C# framework that lets you build reactive full-stack web applications entirely in pure C#—using familiar React-style components, hooks, and declarative patterns. No HTML/CSS/JS required.
