# SignatureInput Widget - Manual Review Checklist

- [ ] Navigate to **Widgets > Inputs > Signature Input** in the Samples app
- [ ] Draw a signature with mouse - verify smooth lines render
- [ ] Click the eraser/clear button - verify canvas resets
- [ ] Test disabled state - drawing should not be possible
- [ ] Test invalid state - verify error icon appears
- [ ] Test different pen colors (Blue, Red) and thickness (1, 4)
- [ ] Verify signature byte[] is captured (check "Signature captured (N bytes)" text)
- [ ] Test on a touch device if available
- [ ] Verify the widget works inside a Form with Field label/description
