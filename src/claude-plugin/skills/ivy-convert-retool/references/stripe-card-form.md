# Stripe Card Form

An embedded payment form to securely collect and submit card information to Stripe. A token is returned that can be used to charge the card.

## Retool

```toolscript
// Stripe Card Form configured in the inspector
stripeCardForm1.stripePublishableKey = "pk_test_abc123..."
stripeCardForm1.submitButtonText // read-only

// Read the generated token after submission
const token = stripeCardForm1.stripeToken

// Event handler: On Stripe Token Created -> run query to charge card

// Scroll into view
stripeCardForm.scrollIntoView({behavior: 'auto', block: 'nearest'});
```

## Ivy

Ivy does not have a dedicated Stripe widget. Use an `Iframe` to embed a Stripe-hosted payment page, or build a custom `ExternalWidget` wrapping Stripe Elements (React).

```csharp
// Simple embed via Iframe (Stripe Checkout hosted page)
new Iframe("https://checkout.stripe.com/pay/cs_test_abc123")
    .Width(Size.Full())
    .Height(Size.Units(100));

// For full Stripe Elements integration, use an ExternalWidget
[ExternalWidget(
    "frontend/dist/StripeForm.js",
    StylePath = "frontend/dist/style.css",
    ExportName = "StripeForm",
    GlobalName = "MyApp_Widgets_StripeForm")]
public record StripeForm : WidgetBase<StripeForm>
{
    [Prop] public string? PublishableKey { get; set; }
    [Prop] public string? SubmitButtonText { get; set; }
    [Event] public Func<Event<StripeForm>, ValueTask>? OnTokenCreated { get; set; }
}
```

## Parameters

| Parameter              | Documentation                                                  | Ivy                                                          |
|------------------------|----------------------------------------------------------------|--------------------------------------------------------------|
| stripePublishableKey   | The Stripe publishable key to generate tokens                  | Not supported (requires ExternalWidget `[Prop]`)             |
| stripeToken            | The token generated by Stripe (read-only)                      | Not supported (requires ExternalWidget `[Event]`)            |
| submitButtonText       | The text displayed on the submit button (read-only)            | Not supported (requires ExternalWidget `[Prop]`)             |
| events                 | Configured event handlers                                      | Not supported (requires ExternalWidget `[Event]`)            |
| hidden                 | Whether the component is hidden from view                      | `.Visible(bool)` on Iframe                                   |
| id                     | The unique identifier (name)                                   | Variable name in C#                                          |
| margin                 | The amount of margin to render outside                         | Not supported (use layout containers)                        |
| style                  | Custom style options                                           | `.Width()`, `.Height()`, `.Scale()` on Iframe                |
| isHiddenOnDesktop      | Whether to hide on desktop layout                              | Not supported                                                |
| isHiddenOnMobile       | Whether to hide on mobile layout                               | Not supported                                                |
| maintainSpaceWhenHidden| Whether to take up space when hidden                           | Not supported                                                |
| showInEditor           | Whether to show in editor when hidden                          | Not supported                                                |
| scrollIntoView()       | Scrolls the component into the visible area                    | Not supported                                                |
| On Stripe Token Created| Event: a Stripe token is created                               | Not supported (requires ExternalWidget `[Event]`)            |
