# Pull Requests

<Ingress>
The Pull Requests app lists all PRs that have been created from completed plans, with links to the associated repositories and cost details.
</Ingress>

## Overview

After a plan is approved and the `MakePr` promptware runs, the resulting pull request appears here. Each entry shows:

- **Repository** — Which repo the PR was created in
- **Plan** — The originating plan with its description
- **Cost** — Total cost incurred for the plan lifecycle
- **Status** — Whether the PR is open, merged, or closed

## Actions

| Action | Description |
|--------|-------------|
| **Open on GitHub** | View the PR in your browser |
| **View Plan** | Navigate to the originating plan |
